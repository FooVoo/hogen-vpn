# Инструкция для пользователя

Четыре инструмента на выбор:

| | MTProxy | VLESS+Reality | Shadowsocks | IKEv2 |
|---|---|---|---|---|
| Что делает | Только Telegram | Весь трафик | Весь трафик | Весь трафик |
| Приложение | Встроено в Telegram | Нужно отдельное приложение | Нужно отдельное приложение | Встроено в iOS, macOS, Windows |
| Настройка | Одно нажатие по ссылке | Импорт по ссылке или QR | Импорт по ссылке или QR | По инструкции или профиль |
| Особенность | Только для Telegram | Самая высокая маскировка | Широкая поддержка | Не нужен сторонний клиент |

---

## Часть 1 — MTProxy (только Telegram)

### Способ А — По ссылке (проще всего)

Получи ссылку вида `https://t.me/proxy?server=...` и просто **нажми на неё** — Telegram откроется и предложит подключиться. Нажми **Подключиться**.

Готово. В верхней части экрана появится значок щита или надпись «Прокси включён».

### Способ Б — Вручную

**Android / iPhone:**
1. Telegram → **Настройки → Данные и память → Тип соединения**
2. **Использовать прокси → Добавить прокси → MTProto**
3. Введи: сервер `<IP>`, порт `2083`, секрет `<СЕКРЕТ>`
4. Сохрани — статус должен стать **Подключён**

**Компьютер (Windows / macOS):**
1. Telegram → **Настройки → Продвинутые настройки → Тип соединения**
2. **Использовать собственный прокси → MTProto**
3. Введи те же данные, сохрани

### Отключить MTProxy

**Настройки → Данные и память → Тип соединения → Прямое подключение**

## Часть 2 — VLESS+Reality (весь трафик, высокая маскировка)

VLESS+Reality работает как полноценный VPN — через него идут браузер, приложения и Telegram одновременно. Трафик маскируется под обычное HTTPS-соединение к известным сайтам.

> ⚠️ **Важно:** Cover-domain сервера меняется автоматически раз в 2 часа. Если профиль перестал работать — зайди на страницу с данными и импортируй свежую ссылку или QR-код.

### Шаг 1 — Установи приложение

| Платформа | Приложение | Где взять |
|---|---|---|
| Android | **v2rayNG** | [GitHub](https://github.com/2dust/v2rayNG/releases) или Google Play |
| iPhone | **Shadowrocket** | App Store (~$3) или **Streisand** (бесплатно) или **V2Box** |
| Windows | **v2rayN** | [GitHub](https://github.com/2dust/v2rayN/releases) |
| macOS | **Hiddify** | App Store или [GitHub](https://github.com/hiddify/hiddify-app/releases) |

### Шаг 2 — Подключись по ссылке или QR-коду

Получи от администратора ссылку вида:
```
vless://...@IP:8443?...#MyVPN
```

**Android (v2rayNG):**
1. Нажми `+` → **Import config from clipboard** (если ссылка скопирована)
   или **Scan QR code**
2. Сервер появится в списке — нажми на него
3. Нажми кнопку запуска (треугольник внизу справа)
4. Разреши VPN-подключение

**iPhone (Shadowrocket):**
1. Главный экран → `+` (вверху справа) → **Scan QR**
   или скопируй ссылку — Shadowrocket подхватит из буфера
2. Нажми на сервер, затем включи тумблер вверху

**iPhone (Streisand):**
1. `+` → **Import** → вставь ссылку или отсканируй QR
2. Включи тумблер

**Windows (v2rayN):**
1. Правой кнопкой на иконку в трее → **Import from clipboard**
   или откройте окно: **Servers → Add server from clipboard**
2. Правой кнопкой на сервер → **Set as active server**
3. **System proxy → Set as system proxy**

**macOS (Hiddify):**
1. `+` → **Add from clipboard** или **Scan QR code**
2. Включи соединение

### Шаг 3 — Проверь

Открой в браузере [2ip.ru](https://2ip.ru) — должен показать IP VPS-сервера, а не твой домашний.

## Часть 3 — Shadowsocks (весь трафик, широкая поддержка)

Shadowsocks — альтернатива VLESS с более широкой поддержкой приложений. Используй шифр `2022-blake3-aes-256-gcm` — убедись, что приложение поддерживает именно этот тип.

### Шаг 1 — Установи приложение

| Платформа | Приложение | Где взять |
|---|---|---|
| Android | **v2rayNG** | Google Play или [GitHub](https://github.com/2dust/v2rayNG/releases) |
| Android | **Shadowsocks** | [Google Play](https://play.google.com/store/apps/details?id=com.github.shadowsocks) |
| iPhone | **Shadowrocket** | App Store (~$3) |
| iPhone | **Potatso Lite** | App Store (бесплатно) |
| Windows | **v2rayN** | [GitHub](https://github.com/2dust/v2rayN/releases) |
| macOS | **Hiddify** | App Store или [GitHub](https://github.com/hiddify/hiddify-app/releases) |

### Шаг 2 — Подключись

Получи от администратора ссылку вида:
```
ss://...@IP:8388#MyVPN
```

**Android (v2rayNG):**
1. `+` → **Import config from clipboard** или **Scan QR code**
2. Выбери сервер → запусти

**Android (Shadowsocks):**
1. `+` → **Scan QR code** или **Import from clipboard**
2. Выбери профиль → кнопка подключения

**iPhone (Shadowrocket / Potatso Lite):**
1. Нажми `+` → вставь ссылку или отсканируй QR
2. Включи тумблер

**Windows (v2rayN):**
1. **Servers → Add server from clipboard** или сканируй QR
2. Правой кнопкой → **Set as active**, затем включи системный прокси

**macOS (Hiddify):**
1. `+` → **Add from clipboard** или **Scan QR code**
2. Включи соединение

## Часть 4 — IKEv2/IPSec (весь трафик, встроенный клиент)

IKEv2 встроен в iOS, macOS и Windows — стороннее приложение не требуется. Используй данные со страницы с параметрами: **сервер**, **логин**, **пароль** и **Pre-shared key (PSK)**.

### iPhone / iPad

1. **Настройки → VPN → Добавить конфигурацию VPN**
2. Тип: **IKEv2**
3. Заполни поля:
   - **Описание:** любое название
   - **Сервер:** IP сервера
   - **Удалённый ID:** IP сервера
   - **Аутентификация пользователя:** Логин
   - **Логин:** из страницы параметров
   - **Пароль:** из страницы параметров
4. Нажми **Готово** → включи переключатель VPN

Если администратор выдал файл `.mobileconfig` — **просто открой его** (AirDrop, файлы, письмо). iOS предложит установить профиль — соглашайся. После установки VPN появится в списке автоматически.

### macOS

**Вариант 1 — Через профиль (рекомендую):**
Открой файл `.mobileconfig` — macOS предложит установить профиль через Настройки системы.

**Вариант 2 — Вручную:**
1. **Системные настройки → VPN → Добавить VPN-соединение**
2. Тип: **IKEv2**
3. Адрес сервера и удалённый ID: IP сервера
4. Аутентификация: **Имя пользователя**
5. Введи логин и пароль

### Android

1. Установи [**strongSwan**](https://play.google.com/store/apps/details?id=org.strongswan.android) из Google Play
2. Нажми `+` → введи данные вручную:
   - **Gateway:** IP сервера
   - **VPN type:** IKEv2 EAP (Username/Password)
   - **Username / Password:** из страницы параметров
3. Или импортируй файл `.sswan` если администратор его выдал: `+` → **Import VPN profile**

### Windows

1. **Настройки → Сеть и Интернет → VPN → Добавить VPN-подключение**
2. **Поставщик VPN:** Windows (встроенный)
3. **Тип VPN:** IKEv2
4. **Адрес сервера:** IP сервера
5. **Тип данных для входа:** Имя пользователя и пароль
6. Введи логин и пароль → **Сохранить** → подключись

> ⚠️ На Windows иногда требуется добавить PSK через командную строку:
> ```powershell
> Add-VpnConnectionTriggerTrustedNetwork -ConnectionName "MyVPN"
> Set-VpnConnection -Name "MyVPN" -AuthenticationMethod EAP
> ```
> Если возникают проблемы — попробуй приложение [strongSwan для Windows](https://www.strongswan.org/) или используй VLESS/Shadowsocks вместо IKEv2.

## Вопросы

Если не подключается — напиши тому, кто дал инструкцию.

- **VLESS** перестал работать → нужно обновить ссылку/QR (cover-domain меняется раз в 2 часа)
- **Shadowsocks** не подключается → убедись, что приложение поддерживает `2022-blake3-aes-256-gcm`
- **IKEv2** не подключается → проверь, что PSK введён правильно; на iOS/macOS лучше использовать `.mobileconfig`-профиль
